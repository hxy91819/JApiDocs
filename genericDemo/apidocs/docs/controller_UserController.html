用户信息

method：GET, POST
url: /api/v1/user/userInfo

参数列表

参数名类型必需描述
userIdnullfalse用户ID

返回结果: Android Code | IOS Code

{
  "userId": "string //用户id",
  "userName": "string //用户名"
}
